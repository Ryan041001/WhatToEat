# 🚀 快速开始 - 微信小程序版

## 📦 文件说明

已完成的代码文件位于 `/miniprogram/` 目录下，包含：

### ✅ 全局文件
- `app.js` - 全局逻辑和数据管理
- `app.json` - 全局配置（页面路由、TabBar）
- `app.wxss` - 全局样式
- `project.config.json` - 项目配置
- `sitemap.json` - 搜索配置

### ✅ 已完成的页面（5个）
1. **首页** (`pages/home/`) - 摇一摇、快捷入口、热门推荐
2. **大转盘** (`pages/spin/`) - Canvas 转盘抽取
3. **卡片滑选** (`pages/swipe/`) - 左右滑动选择
4. **餐厅列表** (`pages/restaurants/`) - 筛选、排序、拉黑
5. **我的** (`pages/mine/`) - 统计、添加餐厅入口

## 🛠️ 使用步骤

### 第一步：打开微信开发者工具
1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开工具，选择"小程序"

### 第二步：导入项目
1. 点击"导入项目"
2. 选择 `/miniprogram/` 文件夹作为项目目录
3. AppID 选择"测试号"（或输入你的 AppID）
4. 项目名称：干饭防纠结

### 第三步：准备 TabBar 图标（可选）
如果需要显示 TabBar 图标，在 `images/tab/` 目录下放置以下图标：

```
images/tab/
├── home.png (未选中)
├── home-active.png (选中)
├── spin.png
├── spin-active.png
├── swipe.png
├── swipe-active.png
├── restaurant.png
├── restaurant-active.png
├── mine.png
└── mine-active.png
```

**图标规格**：81px × 81px，PNG 格式

**临时方案**：如果没有图标，可以在 `app.json` 中删除 `iconPath` 和 `selectedIconPath` 字段，只使用文字标签。

### 第四步：运行项目
1. 点击"编译"按钮
2. 在模拟器中查看效果
3. 切换不同的 TabBar 体验各个功能

## 🔧 数据说明

### 初始数据
应用启动时会自动初始化 6 家默认餐厅：
1. 老妈蹄花火锅（川菜）
2. 一兰拉面（日料）
3. 桥香园米线（米线）
4. 韩式炸鸡（韩餐）
5. 麦当劳（快餐）
6. 云海肴（云南菜）

### 数据存储
- 使用微信小程序的 `LocalStorage` 存储
- 数据修改后自动保存
- 关闭小程序后数据保留

### 清除数据
在微信开发者工具中：
1. 点击右上角"清缓存" → "清除数据缓存"
2. 重新编译，数据恢复为初始状态
