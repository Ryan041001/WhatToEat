## 遇到的问题和解决

1. 问题：玻璃态效果在不同页面的一致性难以保证。
	解决：建立统一的 CSS 变量体系，通过 `theme.css` 集中管理所有玻璃态相关的样式变量，确保各页面使用相同的设计语言。

2. 问题：导航栏在不同页面的适配存在差异。
	解决：重构 Layout 组件，使用 `unified-topbar` 类统一导航栏样式，并通过 CSS 变量控制背景色和边框，实现灵活的主题适配。

3. 问题：视觉层次和深度感的表现不够自然。
	解决：通过精心调整 `backdrop-filter`、`box-shadow` 和透明度参数，配合渐变背景（`glass-atmosphere`），营造出自然的空间层次感。
