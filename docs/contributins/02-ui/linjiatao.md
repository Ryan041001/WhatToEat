## 遇到的问题和解决

1. 问题：设计文档中的文件结构与项目实际目录不一致。  
	解决：按当前仓库目录重新梳理并更新 `design-spec.md` 的文件结构部分。

2. 问题：页面命名与文档命名不统一（如 `SpinWheel`、`CardSwipe`）。  
	解决：在文档中统一采用当前代码命名，避免设计与开发沟通歧义。

3. 问题：设计说明初版信息过细，阅读成本较高。  
	解决：保留关键主干结构，输出精简版目录与核心说明。
