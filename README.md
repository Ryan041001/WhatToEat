## 项目简介
本项目是一款基于微信小程序的"干饭防纠结"微信小程序,目标用户主要为在校大学生及周边职场人群。针对用户日常就餐时"不知道吃什么"的痛点,应用提供了餐厅随机抽取(大转盘/摇一摇)、类似探探卡片的"左滑右滑"选店等趣味防纠结功能。

此外,项目当前阶段以高德 POI 作为餐厅主数据来源,并由后端统一封装地图与位置服务接口。应用仍支持用户提交偏好数据（如拉黑与备注）来做个性化过滤。项目旨在通过轻量级的移动端交互,结合稳定高可用的后端服务,快速解决大众就餐的选择困难症。

## 技术栈(当前方案)
- **前端:** 微信小程序原生开发
- **后端:** Spring Boot (RESTful API) + Spring Data JPA
- **地图与位置服务:** 高德 Web 服务 API（由后端调用并封装）
- **数据库:** MySQL（主要存储用户侧数据，如黑名单、备注、选择历史）
